🎬 ANIMATIONS & INTERACTIONS
═══════════════════════════════════════════════════════════════

TRANSITIONS:
  • Page transitions: Slide (300ms ease-in-out)
  • Modal: Fade + Scale from bottom (250ms)
  • Tabs: Slide indicator (200ms ease)

MICRO-INTERACTIONS:
  • Button press: Scale 0.98 (100ms)
  • Heart like: Scale 1.2 → 1.0 + Flying animation (800ms)
  • Card hover: Lift 2px + shadow (200ms)
  • Pull to refresh: Spring animation

LOADING STATES:
  • Shimmer effect: Gray gradient animation
  • Spinner: Orange color, 40px size
  • Skeleton: Gray 200 background, animated

GESTURES:
  • Swipe: Vertical (video feed), Horizontal (tabs)
  • Tap: Standard tap feedback
  • Long press: Context menu (200ms delay)
  • Pinch to zoom: On images

📝 NOTES FOR FLUTTER IMPLEMENTATION
═══════════════════════════════════════════════════════════════

1. Use Material Design with custom theme for Android
2. Use Cupertino widgets where appropriate for iOS feel
3. Implement custom colors in ThemeData
4. Create reusable widget components (buttons, cards, inputs)
5. Use Provider/Riverpod for state management
6. Video player: video_player package
7. Animations: Use AnimatedContainer, Hero, PageView
8. Bottom nav: Custom BottomNavigationBar
9. Icons: flutter_icons or cupertino_icons
10. HTTP: dio or http package
11. Local storage: shared_preferences
12. Image caching: cached_network_image
